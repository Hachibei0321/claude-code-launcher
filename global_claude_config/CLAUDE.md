# Claude Code 共通設定

## 言語・コミュニケーション

- **日本語で回答**してください
- コードのコメントも日本語で書いてください
- 分からないことがあれば勝手に判断せず質問してください

---

## コーディングスタイル

### Python
- 型ヒントを使用する
- docstringはGoogle形式で書く
- 1ファイル500行を超えたら分割検討

### JavaScript/TypeScript
- モダンなES6+構文を使用
- async/awaitを優先

### 共通
- 変数名・関数名は英語で分かりやすく
- マジックナンバーは定数化

---

## 作業スタイル

### やってほしいこと
- ✅ 変更前に現状を確認
- ✅ 変更箇所を明確に説明
- ✅ エラー発生時はトレースバックを表示
- ✅ 重要な変更は確認を取る

### やらないでほしいこと
- ❌ 確認なしでファイルを削除
- ❌ 動作確認なしで大量変更
- ❌ 存在しないライブラリの使用

---

## デバッグ出力

視認性のため以下の形式を推奨：

```
✅ 成功: 処理完了
❌ エラー: ファイルが見つかりません
⚠️ 警告: 値が空です
📎 INFO: 処理開始
🚀 DEBUG: 変数値 = xxx
```
